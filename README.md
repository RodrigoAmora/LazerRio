# Menu Rio
Menu Rio is an APP for Android that list Options of Leisure, Restaurants, Beaches, Museuns in Rio de Janeiro. <br>
This project run from Android 4.1 <br>

This Porject use as dependencies:
- MaterialDesign;
- GoogleMapsAPI;
- Retrofit 2;
- Daager 2;
- CircleViewImage;
- Picasso;
